The user is presented with two options: one for when AppSheet is installed, and one for when it is not installed. If the user does not have AppSheet, they should install AppSheet from the Play Store. When they open it, they will be taken to the app that you shared (in this case âÄúDriver DispatchâÄù), and a shortcut to the app will be placed on their homescreen.

However, if AppSheet is already installed, tapping the second button will launch AppSheet with the correct app and create a shortcut on the homescreen.
