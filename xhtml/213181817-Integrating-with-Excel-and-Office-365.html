Integrating with Excel and Office 365 – AppSheet

An easy way to build AppSheet apps from Excel is to use the AppSheet add-on found in the Office add-in store.

While this process is mostly seamless, there are some specific behaviors to be aware of. With Office365, if one user has the spreadsheet open for editing, other users and third party apps like AppSheet cannot make updates to the spreadsheet. Consequently, if you intend to build an app that updates the data (as most apps do!), please make sure to close the spreadsheet first. Otherwise, you will see conflict ("409") error messages and the app will not able to save changes to the spreadsheet.
